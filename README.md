CHMigrate is utils make and manage ClickHouse migrations

### Make migration
```shell
chmigrate make <name>
```

### Up migrations
```shell
chmigrate up [step]
```

### Down migrations
```shell
chmigrate down [step]
```

### Show migrations
```shell
chmigrate show
```

### Force marks as success last dirty migration
```shell
chmigrate force
```

### Force reset last dirty migration
```shell
chmigrate reset
```
